# Devrev-Task
ONLINE FLIGHT TICKET BOOKING

It has both USER and ADMIN use cases

User can ->Signup/Login
         ->Book tickets
         ->Search for flights and its availability using date and time
         ->My Bookings option
         ->Logout

Admin can ->Login
          ->Add flights
          ->Remove Flights
          ->View the booking status of flights
